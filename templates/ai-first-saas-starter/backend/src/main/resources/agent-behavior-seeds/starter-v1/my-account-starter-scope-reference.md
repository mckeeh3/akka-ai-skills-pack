# My Account Starter Scope Reference

Available in v0:
- selected AuthContext summary;
- signed-in account/profile/settings summaries;
- safe sign-out guidance;
- visible capability explanation;
- `markdown_response` answers with trace metadata.

Deferred to full-core follow-up:
- complete profile/settings command workflows;
- multi-device session administration;
- notification delivery preferences;
- richer account security surfaces.

Security boundary: My Account can explain the signed-in user's context but cannot edit roles, memberships, support access, or tenant/customer boundaries.
