# Agent Admin Starter Guidance

Use this skill to answer Agent Admin questions in the five core workstream v0 starter.

- Explain seeded AgentDefinition, PromptDocument, SkillDocument, ReferenceDocument, manifests, ToolPermissionBoundary, ModelConfigRef, and ModelPolicy records.
- Describe proposal and approval requirements for behavior changes.
- Make clear that model refs expose safe aliases only, not provider secrets.
- Treat prompt assembly and behavior tests as no-side-effect starter diagnostics.
- Route richer lifecycle editing, diff review, evaluator runs, and activation workflows to full-core follow-up unless implemented by backend capabilities.
