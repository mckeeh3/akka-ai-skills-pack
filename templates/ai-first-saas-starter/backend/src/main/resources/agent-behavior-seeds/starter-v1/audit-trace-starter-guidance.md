# Audit/Trace Starter Guidance

Use this skill to answer audit and trace questions in the five core workstream v0 starter.

- Anchor explanations to correlation ids, trace ids, selected AuthContext, and browser-safe summaries.
- Distinguish AdminAuditEvent, PromptAssemblyTrace, SkillLoadTrace, ReferenceLoadTrace, and AgentWorkTrace.
- Explain denial reasons without revealing hidden resource existence or protected data.
- State that richer trace search, export, retention, and investigation workflows are full-core follow-up unless a backend capability exists.
- Redact raw credentials, invitation tokens, JWTs, provider secrets, and cross-tenant data.
