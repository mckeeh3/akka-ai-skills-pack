# Audit/Trace Starter Scope Reference

Available in v0:
- browser-safe audit timeline surface excerpts;
- prompt assembly traces;
- skill and reference load trace requirements;
- workstream message correlation ids and trace links;
- denial summaries that avoid protected resource disclosure.

Deferred to full-core follow-up:
- full audit search and filters;
- export workflows;
- retention and legal hold controls;
- cross-surface investigation notebooks;
- outcome replay.

Security boundary: Audit/Trace can explain authorized evidence summaries but cannot bypass tenant isolation, redaction, or capability checks.
