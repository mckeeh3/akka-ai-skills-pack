# Current-intent app-description templates

Use these small templates when bootstrapping or repairing the canonical current-intent graph shape described in `docs/current-intent-model.md`.

Copy only the files needed for the selected scope, replace `_domain` and `_workstream` placeholders with real ids, and keep the app-description source-alignment artifact current whenever feature-bearing graph files or implementation files change.

Included templates:

```text
domains/_domain/workstreams/_workstream/realization/source-alignment.md
```
