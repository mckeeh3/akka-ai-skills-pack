# Workers: Governance/Policy

Worker bindings for the Governance/Policy workstream. These files preserve the current skills-pack worker → harness → actor adapter → governed tool → capability chain.

- `governance-policy-human-operators.md`: role-variant human worker binding for SaaS owner admins, tenant admins, auditors, and scoped support users.
- `governance-policy-functional-agent-worker.md`: model-backed workstream assistant behind `governance-policy-agent`.
- `governance-policy-system-worker.md`: deterministic policy catalog/effective-decision/history/trace worker.
