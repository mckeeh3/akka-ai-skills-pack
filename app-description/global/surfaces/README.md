# Global Surfaces

Reusable surface and shell definitions for the core starter current-intent graph.

- `workstream-shell.md` is the comprehensive global contract for the authenticated AI-first workstream shell: layout, rail, selected context, composer, stream/surfaces, actions, realtime, recovery, security, accessibility, and tests.
- `ui-style-and-runtime-contracts.md` points generated and maintained UI work to the active frontend source paths, style tokens, runtime rules, and non-inference boundaries.
- `foundation-surface-patterns.md` defines reusable foundation surface patterns.
