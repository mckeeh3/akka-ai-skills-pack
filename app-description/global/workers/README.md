# Global Workers

Reusable worker definitions for the core starter current-intent graph. Workstream files bind these shared workers to local surfaces, agents, governed tools, policies, traces, tests, and realization evidence.

Current shared worker artifacts:

- `foundation-workers.md` — cross-workstream foundation worker taxonomy, harnesses, actor adapters, authority boundaries, trace obligations, and failure behavior for human, functional-agent, deterministic system, provider-boundary, onboarding/integration, and audit/projection workers.
