System Message Surfaces

System feedback is a structured surface with recovery actions, not a toast-only string.

Denied

Validation

Enter a policy name and choose an approval threshold before saving.

Approval required

This action expands authority and must be approved by an organization admin.

Stale

This surface used older projection data. Refresh before taking action.

No-op
