# Task Briefs

This directory is reserved for detailed per-task briefs when a pending task needs more guidance than the entry in `../pending-tasks.md`.

Initial migration tasks are small enough to run from:

1. `../README.md`
2. the relevant sprint spec under `../sprints/`
3. the matching backlog under `../backlog/`
4. the selected entry in `../pending-tasks.md`

If a future task becomes broad or ambiguous, add a focused task brief here using the existing migration-package pattern:

```text
tasks/<sprint-slug>/<task-number>-<task-slug>.md
```

Keep task briefs scoped to one fresh harness session and one git commit.
