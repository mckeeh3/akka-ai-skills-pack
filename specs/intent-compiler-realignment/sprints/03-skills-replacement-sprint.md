# Sprint 03: Active Intent-Processing Skills Replacement

## Objective

Replace or realign active user-intent processing skills so they route flexible user input through the intent compiler model.

## Scope

- Intake/router/normalization skills.
- App-description capture skills.
- Change/reconciliation/readiness/generation skills.
- PRD/spec/backlog/pending-question/pending-task planning skills.
- High-level router skills that introduce AI-first SaaS/workstream/capability semantics.

## Acceptance criteria

- Active skills identify whether they capture, refine, realize, validate, or repair intent.
- Skills write/read the new app-description intent graph structure.
- Skills avoid old flat/global-only app-description assumptions.
