# Sprint 03: Agent Admin and Hybrid Runtime

## Goal

Turn the governed runtime agent doctrine into production-shaped implementation guidance and reference assets for Agent Admin and hybrid Akka agent invocation.

## Outcomes

- Agent Admin covers `AgentDefinition`, prompts, skills, manifests, tool boundaries, model refs, seed imports, test console, lifecycle, and traces.

- Gaps in governed model config and side-effecting/component/MCP tool enforcement are addressed or queued with narrow follow-ups.
