# AI-First Skills Pack Migration Task Briefs

This directory contains physical task briefs for migration work that needs a focused fresh Pi session.

Sprint 1 task briefs cover the initial doctrine pivot. Sprint 8 task briefs cover the first executable AI-first supplies autopilot reference slice. Sprint 10 task briefs plan the authenticated starter core app foundation derived from the user-auth onboarding PoC. Other sprint briefs may be created or refined as discoveries require.

## Rules

- Execute one task brief per fresh context.
- Update `../pending-tasks.md` before finishing.
- Make one git commit for the selected task before marking it `done`; include only that task's intended changes and the queue-status update.
- Record the commit hash in `../pending-tasks.md` task notes when possible.
- Do not combine adjacent task briefs.
- Do not implement source changes outside the selected task's explicit scope.
