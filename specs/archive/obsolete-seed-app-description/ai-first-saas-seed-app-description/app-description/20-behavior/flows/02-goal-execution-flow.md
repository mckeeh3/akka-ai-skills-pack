# Flow: Goal Execution

1. Intent author creates a goal with objective, constraints, and success criteria.
2. Coordinator agent drafts an execution plan and identifies permissions, data/tool needs, and approval gates.
3. Supervisor reviews the plan when required by policy.
4. Workflow launches approved work and records trace context.
5. Specialist agents or human tasks execute bounded steps.
6. Exceptions pause or block the workflow and create decision cards.
7. Completion records work result, trace summary, and outcome placeholders.
8. Outcome owner can later attach metrics and feedback.
