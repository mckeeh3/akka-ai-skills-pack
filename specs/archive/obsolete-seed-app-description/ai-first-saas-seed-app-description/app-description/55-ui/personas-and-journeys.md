# Personas and Journeys

- tenant admin:
  - invite users, assign roles, configure tenant settings, review governance posture
- supervisor:
  - create goals, review plans, launch work, monitor progress, handle exceptions
- reviewer:
  - process decision cards with evidence, risk, confidence, and alternatives
- policy owner:
  - review policy proposals, simulate/replay expectations, approve policy commits
- auditor:
  - search traces and explain who/what/why/how-authorized
- member:
  - use assigned tenant capabilities and view relevant work
