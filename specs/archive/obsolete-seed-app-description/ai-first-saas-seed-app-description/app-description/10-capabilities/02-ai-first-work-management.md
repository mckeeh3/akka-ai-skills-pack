# Capability: AI-First Work Management

- capability-id: ai-first-work-management
- purpose:
  - provide generic durable objects and flows for delegating human objectives to bounded agent-assisted execution
- in scope:
  - goals and objectives
  - execution plans
  - tasks and agent assignments
  - progress, pause/resume, completion, and cancellation
  - tool-use requests where permissioned
  - exception escalation to humans
- out of scope:
  - domain-specific vertical work modules
- primary actors:
  - intent author
  - supervisor
  - agent coordinator
  - specialist agent
  - exception handler

  - `../15-operating-model/goals-and-objectives.md`
