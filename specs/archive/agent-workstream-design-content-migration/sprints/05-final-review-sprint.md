# Sprint 5: Final Review

## Goal

Run a final consistency review across design-related docs, skills, and examples and record migration completion.

## Scope

- Search for remaining page-first, screen-first, CRUD-first, chatbot-bolt-on, and inconsistent terminology drift.
- Verify the design review checklist is linked from the appropriate routing docs and skills.
- Verify `12-workstreams/` and `55-ui/` boundaries are consistent.
- Verify generation/readiness guidance routes generated UI to canonical workstream references.
- Record completion summary.

## Out of scope

- New broad architecture changes discovered during review; create follow-up tasks instead.

## Done criteria

- All migration tasks are done or explicitly superseded.
- Completion summary records what changed, what was intentionally left alone, and any follow-up queue.
