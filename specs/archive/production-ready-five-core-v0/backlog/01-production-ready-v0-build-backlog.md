## Acceptance summary

A freshly scaffolded app is production-ready v0 when:

- `mvn test`, frontend tests/typecheck/build, and starter fullstack validation pass;
- real provider env can be configured through backend-only variables;
- local authenticated bootstrap user can submit a message to each of the five core workstreams;
- each response is generated by the real model-backed governed workstream agent path;
- missing provider configuration fails safely and actionably;
- messages and traces are durable and visible after refresh/reload;
- built frontend contains no backend secrets;
- pack build succeeds and installed source pack can scaffold the app.
