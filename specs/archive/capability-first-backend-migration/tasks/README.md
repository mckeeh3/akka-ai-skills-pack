# Capability-First Migration Task Briefs

This directory is reserved for detailed physical task briefs when a pending task needs more instruction than the queue entry provides.

For now, use:

1. `../README.md`
2. the relevant `../sprints/*.md`
3. the matching `../backlog/*.md`
4. the selected entry in `../pending-tasks.md`

If a future task becomes too broad or ambiguous, add a focused task brief here and update `../pending-tasks.md` to reference it.
