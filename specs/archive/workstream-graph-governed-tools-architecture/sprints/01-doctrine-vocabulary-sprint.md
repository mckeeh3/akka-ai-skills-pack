## Acceptance criteria

- Core docs define dashboard as a role-specific attention/work surface.
- Surface graph and internal workstream agent graph are first-class decomposition concepts.
- Governed-tools are defined as executable semantic operations inside capability files and surface/action maps.
- Qualified terms are used where ambiguity matters: governed-tool, browser-tool, agent-tool, internal-tool.
- Capability remains a product-level grouping/ability, not erased.
